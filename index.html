How to deploy this app
This is a single HTML file — zero build step, zero dependencies. Host it anywhere that serves static files.
Option 1 — Vercel (Recommended)
1
Create a GitHub repo
Go to github.com/new, create a new repo, and upload meeting_english_db.html. Rename it to index.html.
2
Import to Vercel
Go to vercel.com/new, click "Import Git Repository", select your repo. No config needed — click Deploy.
3
Done!
You'll get a free URL like your-app.vercel.app. Every push to GitHub auto-redeploys.
Option 2 — Netlify Drop
1
Rename & drag-drop
Rename the file to index.html, then go to app.netlify.com/drop and drag the file onto the page.
2
Live in seconds
Netlify gives you a free URL instantly. No account required for the first deploy.
Option 3 — GitHub Pages
1
Push index.html to a repo
Create a repo named your-username.github.io and push index.html to the main branch.
2
Enable Pages
Go to repo Settings → Pages → Source → "Deploy from branch". Your site will be live at your-username.github.io.
⚠️ Important Note
Since this app uses localStorage, each browser/device has its own independent database. Use the Export → Import workflow in the Data tab to sync between devices.
⚠️
Clear all data?
